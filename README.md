# ✦ Idealog

A fast, interactive, and completely browser-based project manager. Idealog combines standard spreadsheet functionality with dynamic visualizations to help you track task hierarchies, view custom properties, and follow git commit history paths.

## Key Features

- **📊 Excel-Style Task Sheet**: Create tasks, add custom parameter columns, assign owners, and manage statuses dynamically.
- **Pivot Table**: Slice, dice, and group tasks by multiple dimensions (Project, Status, Assignee, System, Subsystem, Dates) with HTML5 drag-and-drop column sorting.
- **⚡ Interactive Highlights**: Click on any Pivot Table cell parameter to instantly trace its parent and child connection path all the way down to individual tasks.
- **⏰ Smart Highlight Fading**: Recurrently updated rows light up with a subtle greenish tint that fades away over a custom user-defined day interval, making it easy to track shifting tasks.
- **⇆ Git Board**: Visual interactive commit graph tracker with integrated diff panels showing changes over time.
- **💾 Local Persistence & Sync**: Saves automatically to local storage, exports/imports custom paste arrays, and syncs directly to Google Sheets via push/pull endpoints.
