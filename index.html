Import Sheet Data

Paste tab-separated values (TSV) or copy/paste directly from Excel/Google Sheets. The first row should contain headers. If a matching header name exists, values will fill those columns; otherwise, new custom columns will be created.

Import Object Lists Data

Paste a spreadsheet table to extract unique values for **Assignees**, **Systems**, **Subsystems**, and **Projects** simultaneously. The parser will detect columns matching these names (case-insensitive) and import all unique values.

Connect Idealog directly to your personal **Supabase** database instance. Data will synchronize automatically in the background across all devices.
